commando 是什么

Configuration 归你所有，我们让它可以分享。

三个原语，没有 SaaS 绑定。数字搭档跑在你已经在用的任意 AI 工具上。

01 — Charter

章程

数字搭档的身份、风格、ICP、红线。每次调用自动注入。一份 markdown 文件躺在你的仓库里。

02 — Skills

离散任务

写一篇笔记、复盘一场会议、扫一遍竞品。被 Schedule 触发或你 IM 召唤。每个 Skill = 一份 SKILL.md。

03 — Schedule

排班调度

自动翻译成 launchd / systemd。操作系统跑你的搭档，不是我们。

三个真实的 Configuration

数字搭档到底长什么样？

三份不同领域的 Charter 节选，展示这个工具的覆盖范围。每个 Charter 就是几百行 markdown，定义了这个搭档是谁。

角色 · 增长合伙人

独立产品创作者的内容 + 用户洞察搭档

已经为 LeMingle 持续服务 6+ 个月——本仓库的标准参考 Configuration。

"我是 [产品名] 的增长合伙人。风格：朴实、数据先行、不端着、绝不用营销腔。
红线：AIGC 手改率 ≥ 20%、小红书外链零容忍、营销密度 ≤ 20%。"
morning_sense 每天 08:00  ·  xhs_draft 周二/五 11:00  ·  weekly_data_review 周五 16:00  ·  weekly_reflection 周日 19:00

角色 · 理财顾问

始终在线的账户守望者

展示强红线机制如何让数字搭档比通用 AI 助手更安全。

"我是你的私人理财顾问。红线：绝不替你做任何交易；绝不预测市场；
绝不推荐高风险产品；季度报告必须包含一节'我可能错在哪'。"
daily_portfolio_health 每天 07:00  ·  monthly_tax_check 每月 1 日 09:00  ·  quarterly_rebalance_review 每季度末

角色 · 求职陪跑导师

跳槽 3 个月的高强度陪练

演示有时限的陪伴关系——Charter 里写明的"失败标准"。

"我是你的跳槽教练。风格：温和但不放水。红线：绝不帮你写虚假履历；
绝不替你回避真实弱点；3 个月内毫无进展我会直接告诉你。"

为什么用 commando

归你所有，不是租来的。

Runtime 是商品，Configuration 是护城河。我们故意不卷 Runtime——把全部精力放在让 Configuration 可分享、可版本化、归你所有。

commando ChatGPT / GPTs LangChain Zapier + LLM
归你所有 ✓ 文件在你仓库里 ✗ 锁在 OpenAI ✓ 代码在你仓库 ✗ 锁在 Zapier
自由选 LLM ✓ Claude / GLM / Kimi / 任意 ✗ 只能 OpenAI ⚠ 每换一家都改代码 ⚠ 只能用它的集成
用系统定时器 ✓ launchd / systemd ✗ ⚠ 自己写 ✓ 用它的 cron
输出到你的工具 ✓ 飞书 / Notion / IM ✗ 只在 ChatGPT 里 ⚠ 自己接 ✓ 用它的集成

有引导上手 ✓ 25 分钟对话 ✗ 自己想 ✗ 自己想 ✗ 自己想

开始使用

你的数字搭档明早就上岗。

装好、onboard、go-live。第二天早上 8 点，搭档跑第一个任务，飞书弹卡片找你。

$ curl -fsSL https://raw.githubusercontent.com/Frank-Pu/commando/main/install.sh | sh

# step 2 · 25 分钟对话 → 产出 ./my-agent/

# step 3 · 填好 Skill body + 接 IM + 装定时调度
